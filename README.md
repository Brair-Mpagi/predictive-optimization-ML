Repository containing Python scripts for data science and operational research. Features Linear Programming (LP) optimization models built with PuLP (cost minimization, profit maximization, 3D constraints) and predictive modeling via linear/logistic regression.


### Operations Research & Optimization (`model.pdf`)
A collection of 8 Linear Programming (LP) optimization problems modeled with `PuLP` and visualized dynamically using `Matplotlib` (2D and 3D).

* **Resource & Cost Optimization (Problems 1–4):** Minimizing hardware infrastructure costs (CPU, memory, and storage allocations) against multi-tenant requirements.
* **3D Multivariable Minimization/Maximization (Problems 5–6):** Industrial operations scripts optimizing raw materials, labor, and budget investments in 3D coordinate space.
* **Diet & Production Management (Problems 7–8):** Classic operations research problems solving nutritional cost minimization and factory profit maximization.
